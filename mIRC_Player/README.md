----------------------- Sobre o mIRC Player -----------------------

--> Descrição: Esse addon é um player de música, suporta todos os formatos compatíveis
                           com o mIRC, não usa DLL nenhuma!

--> Como instalar:
-    1º: Abra teu script
-    2º: E digite:
- /load -rs "(pasta onde se encontra o addon)\mIRC_Player\mIRC_Player.mrc"
- Obs.: Coloque as aspas
-
-----------------------------------------------------------------------------------
