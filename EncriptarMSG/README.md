----------------------- Sobre Como Encriptar a Mensagem -----------------------

--> Descrição: Com esse Addon, toda letra que você escrever será trocada
                          por símbolos animados e coloridos que simula a letra, divertido ;)

--> Como instalar:
-    1º: Abra teu script
-    2º: E digite:
- /load -rs "(pasta onde se encontra o addon)\EncriptarMSG\encriptar.mrc"
-Obs.: Tem que por as aspas
