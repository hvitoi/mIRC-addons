----------------------- Sobre Risadas Com Contadores e Cáculos -----------------------

--> Descrição: Com esse Addon, suas risadas serão aletórias, terão contador,
                           cálculo do contador e um som de uma risada, 
                           basta apertar F3, leia as Instruções. ;)

--> Como instalar:
-    1º: Abra teu script
-    2º: E digite:
- /load -rs "(pasta onde se encontra o addon)\Visualizador\visualizador.mrc"
-Obs.: Tem que por as aspas
